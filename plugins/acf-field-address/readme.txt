== Description ==

Refactored Advanced-custom-fields-address-field-add-on (by Brian Zoetewey) for AFC 4.0 and above

= Compatibility =

This add-on will work with:

* Advanced Custom Fields version 4 and up

***DOES NOT WORK WITH VERSIONS BELOW 4***

== Installation ==


= Plugin =

2. Activate the plugin via the Plugins admin page
